# Bank Management System

Bank Management System is an academic project developed to understand
the working of basic banking operations using programming concepts.

## Features
- Account creation
- User login authentication
- Balance inquiry
- Deposit and withdrawal of money
- Simple transaction handling

## Technologies Used
- Java
- Object Oriented Programming (OOP)
- Basic Data Structures
- File handling / Database (if applicable)

## Project Type
Academic Project

## Learning Outcomes
- Understanding real-world banking workflows
- Improved knowledge of Java and OOP concepts
- Hands-on experience with logical problem solving
- Basic understanding of backend system design

Completed / In Progress
